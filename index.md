# GeoAI with Python

A hands-on guide to geospatial artificial intelligence for Earth observation, remote sensing, and spatial analysis.

## About This Book

This book teaches you how to apply modern AI techniques to geospatial data using Python. From downloading satellite imagery to building QGIS plugins, each chapter builds practical skills through working code examples using the [geoai](https://github.com/opengeos/geoai) package and the broader Python geospatial ecosystem.
